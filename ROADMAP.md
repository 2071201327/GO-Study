# Go 学习路线图

通过标记完成任务来跟踪你的进度 (例如 `[x]`)。

## 第一阶段：基础 (Foundation)
- [ ] **01-hello-world**: 基础语法，验证环境。
- [ ] **02-variables-types**: 字符串，整数，布尔值，零值。
- [ ] **03-control-structures**: If/Else, Switch, For 循环。
- [ ] **04-functions**: 多返回值，命名返回值。

## 第二阶段：数据结构 (Data Structures)
- [ ] **05-arrays-slices**: Append, Range, 切片操作。
- [ ] **06-maps**: Map 的增删改查。
- [ ] **07-structs**: 定义，初始化，嵌套。

## 第三阶段：代码组织 (Code Organization)
- [ ] **08-methods**: 值接收者 vs 指针接收者。
- [ ] **09-interfaces**: 接口实现，类型断言。
- [ ] **10-packages**: 导出 vs 未导出，包导入。

## 第四阶段：并发 (Concurrency)
- [ ] **11-goroutines**: 启动 Go 协程，WaitGroups。
- [ ] **12-channels**: 无缓冲 vs 缓冲通道，Select 语句。

## 第五阶段：Go Web (目标)
- [ ] **13-http-server**: 基础 Handler, `http.ListenAndServe`。
- [ ] **14-json**: JSON 序列化/反序列化。
- [ ] **15-routing**: 路由处理。
- [ ] **16-middleware**: 日志，鉴权模拟。
