# 练习 05: 数组与切片

## 目标
掌握数组与切片（Slice）的区别，以及切片的常用操作。

## 指南

1.  创建 `05-arrays-slices` 目录。
2.  创建 `slices.go`。
3.  实现函数：
    - `CreateSlice()`: 返回一个包含 1, 2, 3 的整数切片。
    - `AddElement(s []int, n int) []int`: 使用 `append` 向切片添加元素。
    - `GetSubSlice(s []int, start, end int) []int`: 返回从 `start` 到 `end` (不包含) 的子切片。
