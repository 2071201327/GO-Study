# 练习 06: Maps (映射)

## 目标
学习 Map 的创建、读写和删除。

## 指南

1.  创建 `06-maps` 目录。
2.  创建 `maps.go`。
3.  实现函数：
    - `CreateMap()`: 返回一个 `map[string]int`，包含 {"apple": 5, "banana": 10}。
    - `DeleteEntry(m map[string]int, key string)`: 从 map 中删除指定 key。
    - `GetScore(m map[string]int, name string) (int, bool)`: 获取分数，如果不存在返回 false。
